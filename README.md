## 介绍

zckui-uniapp 组件库, 使用 Vue 技术栈开发小程序应用，开箱即用，拥有丰富的业务组件。

## 特性

- 🚀 n+ 高质量组件，覆盖移动端主流场景
- 💪 支持一套代码同时开发多端
- 🍭 支持按需引用
- 💪 支持 TypeScript
<!-- - 📖 基于京东 APP 10.0 视觉规范 -->

<!-- - 💪 支持动态定制主题 -->
